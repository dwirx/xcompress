# Changelog

All notable changes to xCompress will be documented in this file.

## [0.1.0] - 2026-06-29

### Added

- Added a local compression history modal with output folder reveal actions.
- Added global app settings for restoring the queue and auto-selecting newly added files.
- Added reset actions for compression settings, queue data, history, and local app data.
- Persisted app preferences and recent compression history in local storage.

### Changed

- Made completed previews and preview backgrounds open the large preview directly.
- Turned the file selection indicator into an accessible clickable control.

### Fixed

- Improved hover and pointer behavior for file selection controls.
